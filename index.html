This a small app I built that graphically tracks trending stock tickers in real-time for a single day. It tracks the number of times a specific stock ticker is mentioned on Twitter, Stocktwits and IRC. It was built using Python, Tornado, Flask, D3.JS, and Twisted. Building out stock mentions RESTful API. Trading Dashboard coming soon using Dashing.
